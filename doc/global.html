overview()

This function gives a high-level overview of the program by reading out comments, function definitions, and for loops. As each line is read out, the cursor is positioned at the start of that line. Pressing any key cancels the overview, and the programmer can start editing the program from the next line. In addition, the overview starts from the current position of the cursor within the program.
